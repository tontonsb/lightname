# lightname

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Pre-cache the predefined geodata

```sh
npm run cache-geodata
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

Or with precaching of geodata:

```sh
npm run cache-geodata
npm run build
```
